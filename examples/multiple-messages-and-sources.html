Explanation

This page demonstrates setting the content of multiple target elements, using multiple sources.

Each row in the table loads a randomly-selected message into each of its cells. The source of the message is determined by the column; there is one column for each source.

The three sources used are described in the “multiple sources” example.

In each column, there should be several different messages from a source, all randomly selected and randomly ordered.
