Explanation

This page demonstrates how message sources and message IDs interact.

Each cell in the table loads a randomly-selected message. The message source is determined by the column. The three sources used are described in the “multiple sources” example.

The row determines which message ID will be used, if any:

  • Cells in the “No ID attribute” rows do not have the data-saria-random-message-id attribute at all.
  • Cells in the “Empty ID” rows have the data-saria-random-message-id attribute, but it is empty (or whitespace-only, which is the same as empty).
  • Cells in rows with an ID value all have the data-saria-random-message-id attribute set to that value.

Wherever the source and ID are the same, and the ID is present and not empty, the cell content should be the same.
