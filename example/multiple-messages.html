Saria’s random message loader program: Multiple messages

Explanation

This page demonstrates setting the content of multiple target elements, using a single source.

There are eight <li> elements, all the same: 

<li data-saria-random-message-src="data/alpha.txt"></li>

Each should be filled with a randomly-selected message with the form “Message [<b>x</b>]”, where x is a lower-case letter.

Each list item should contain a different message, and the messages should be randomly-selected and randomly-ordered.
