Saria’s random message loader program: HTML preloading

Explanation

This page demonstrates using HTML preload hints to load the messages file earlier.

In the page <head>, there is a <link> element requesting the messages source file be preloaded: 

<link rel="preload" href="data/quotes.txt" as="fetch" crossorigin="anonymous"/>

In a supporting browser, you can observe the messages file being loaded concurrently with the script, before it even has a chance to execute, meaning the messages will already be available when the script finally needs them.
