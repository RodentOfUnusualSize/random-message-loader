Saria’s random message loader program: Asynchronous loading

Explanation

This page demonstrates using Saria’s random message loader program with asynchronous loading.

The script is loaded with the following element in the page <head>: 

<script src="random-message-loader.js" async=""></script>

Note the async="" attribute. This is the only difference from the basic example.

Loading the script asynchronously can make a page more responsive. The page will not wait for the script to execute; it will continue loading. Meanwhile, the script can be loaded in parallel with the page, and other resources (style sheets, images, etc.).
