Saria’s random message loader program: Basic usage

Explanation

This page demonstrates the most basic usage of Saria’s random message loader program.

The script is loaded with the following element in the page <head>: 

<script src="random-message-loader.js"></script>

A <figure> element is used to receive the content: 

<figure data-saria-random-message-src="data/quotes.txt"></figure>

The messages file, located at the URL data/quotes.txt contains a collection of quotes, marked up as HTML:
