Explanation

This page demonstrates using multiple message sources.

The table contains three rows, one for each source. The cell in the result column of each row has the form: 

<td data-saria-random-message-src="source"></td>

where source is the URL of a source.

Each source has different message content:

Source number Source URL Message format
1 data/alpha.txt “Message [<b>x</b>]”, where x is a lower-case letter
2 data/number.txt “Message #NNNN”, where NNNN is a 4-digit number (range [0000–9999], inclusive)
3 data/emoji.txt Emojis

The result cell of each row should contain a randomly-selected entry from the source.
